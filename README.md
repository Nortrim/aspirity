# Инструкция запуска

## Установка

Необходимо установить пакеты для клиентской и серверной частей приложения 
#### `cd client`
#### `npm i`
#### `cd ..`
#### `cd server`
#### `npm i`
Или запустить 
#### install.sh

## Запуск
### Клиент
#### `cd client`
#### `npm run start`
Или
#### run-client.sh
### Сервер
#### `node server/bin/www`
Или
#### run-server.sh
